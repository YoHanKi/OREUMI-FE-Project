# subscribe-hodu-front-end-project
## ✍기술스택
**HTML , CSS , JavaScript**

### 🐕개요
> 호두(고양이) 구독 홈페이지 프런트엔드 제작 프로젝트이다.
> 
> **카카오맵 API**와 **Picsum 홈페이지 이미지**를 사용하였으며, JS를 이용한 **무한 스크롤 이미지(스로틀링)** 구현을 중점으로 작성하였다.
